# Fake Image Detection using Deep Feature Fusion and Transfer Learning

## Overview

Fake images generated by modern AI systems and image manipulation techniques have become increasingly difficult to distinguish from authentic images. This project explores multiple state-of-the-art deep learning architectures for fake image detection and evaluates their effectiveness through feature extraction, transfer learning, and feature fusion strategies.

The objective is to develop a robust computer vision pipeline capable of distinguishing real and AI-generated images while comparing the performance of different pretrained architectures.

---

## Key Features

* Transfer Learning using pretrained CNN architectures
* Deep Feature Extraction
* Binary Classification of Real vs Fake Images
* Comparative Analysis of Multiple Architectures
* Feature Fusion Experiments
* Performance Evaluation using Accuracy, Precision, Recall, and F1-Score

---

## Models Evaluated

### ResNet50

A deep residual neural network used for extracting high-level visual representations while mitigating vanishing gradient problems.

### EfficientNet-B3

A parameter-efficient architecture that balances network depth, width, and resolution for improved classification performance.

### ConvNeXt

A modern convolutional architecture inspired by Vision Transformers while retaining CNN efficiency.

### Fusion Models

Feature-level fusion combining representations from multiple architectures to improve detection robustness and classification accuracy.
